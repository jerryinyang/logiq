# Logiq

This is a code bundle for Logiq. It's based on default Vite project. The original design is available at https://app.flowstep.ai/file?activeFileId=2d4d9a78-e004-48bc-b2a2-f0ab203acce2.

## Running the code

Run `npm i` and `npm run setup` to install the dependencies.

Run `npm run dev` to start the development server.
